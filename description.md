# Project 01

## Purpose
Connect smartphones to play a game with your friends. You are walking in the woods and need to find a rift in space. The game masters hidden among you are secretely guiding you through the dangerous journey.

## Features
- Connect up to 20 Android smartphones with each other via hotspot wi-fi
- Assign 1 or more smartphones as 'game masters' (admins) that can play videos on all connected smartphones from a central game playlist, turn screen off, deactivate torch
- Game masters can determine a password that other players need to enter to join the game session
- Main screen during a started game session is entirely blue
- Main screen during a started game session displays a connectivity indicator to the current game session
- Game masters can upload videos into the game playlist and manage them (remove, change order)
- Uploaded videos are synchronized to connected smartphones local storage, so that video playback is possible even with slow or intermittent network connection
- App connection to other devices is maintained even with screen off
- Game masters can control video playback (next, previous) with bluetooth remote control
- Video playback pauses after each video in the playlist, can be resumed by input from game masters
- Game masters and players have the same UI during the game (game masters need to be undercover), game masters button to resume video playback needs to be invisible in the game UI
- Manual installation of the App via APK

## Target Audience
Private group of up to 20 people
